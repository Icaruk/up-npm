# up-npm

CLI tool written in Go to review and update your NPM dependencies, easy and fast.

![](https://i.imgur.com/MzzR05S.png)

# Usage

Go where your `package.json` is located and run:

```bash
updatenpm [flags]
```

Flags:

- -h, --help      help
- -v, --version   version
- -d, --dev       Update dev dependencies
